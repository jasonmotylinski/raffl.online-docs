# Editing a Raffle

You can modify an existing raffle's details at any time.

## Accessing the Edit Page

1. In the top navigation bar, click **Raffles**.
2. Find the raffle you want to edit in the list.
3. Click on the raffle name to open the edit page.

## Editable Fields

You can update the following fields:

- **Event Name** – Name of the event associated with the raffle.
- **Raffle Name** – A short name describing the item being raffled.
- **Description of Raffle** – A more detailed description of the raffle item.
- **Ticket Cost** – Price per ticket.
- **Max Entries** – Maximum number of tickets available.
- **Unique URL Name** – Custom URL for the raffle page.
- **Layout** – Select a display template for your raffle page.

## Saving Changes

Click **Save** to apply your changes. The updates will be reflected immediately on the public raffle page.
