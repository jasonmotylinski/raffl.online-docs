# Viewing Your Raffles

## Raffle List

To view all your raffles:

1. In the top navigation bar, click **Raffles**.
2. You'll see a list of all raffles you've created.

The list shows:

- **Raffle Name** – Click to edit the raffle.
- **Event Name** – The associated event, if any.
- **URL** – The public URL where participants can view and purchase entries.
- **Status** – Whether the raffle is active.

## Pagination

If you have many raffles, the list is paginated with 10 raffles per page. Use the pagination controls to navigate between pages.

## Public Raffle Page

Each raffle has a public URL that participants can visit to:

- View the raffle details and description.
- See which entries are available, pending, or purchased.
- Purchase available entries via PayPal.

The URL follows the format: `raffl.online/raffle/[your-unique-url-name]`

## Entry Display

On the public raffle page, entries show:

- **Available entries** – Can be claimed and purchased.
- **Pending entries** – Currently being purchased (5-minute hold).
- **Purchased entries** – Show the buyer's name from PayPal.
- **Sponsored entries** – Show the sponsor name you assigned.
